=======================
native nxos salt-minion
=======================

This is a native salt-minion docker container for Cisco Nexus 9k.

Setup
-----

To Run:

.. code-block::

    git clone 
    cd nxos-native-salt-minion
    docker-compose up

To Build:

.. code-block::

    docker build -t salt-minion .


Docker Hub:

.. code-block::

    docker pull sh0x/nxos-native-salt-minion
